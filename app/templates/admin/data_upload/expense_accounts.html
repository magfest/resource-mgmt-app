Basic Information

Column Required Accepted Names Notes
Name Yes name, account_name, expense_account Display name

Column Required Accepted Names Notes

Spend Type Settings

Column Required Accepted Names Notes
Default Spend Type No spend_type, default_spend_type Code or name (e.g., DIVVY, BANK, HOTEL_FEE)

Allowed Spend Types No allowed_spend_types, spend_types Comma-separated spend type codes (e.g., BANK,DIVVY,CHECK). Use with ALLOW_LIST mode.

Column Required Accepted Names Notes

Column Required Accepted Names Notes
