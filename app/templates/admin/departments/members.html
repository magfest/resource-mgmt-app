Work Type Access

Each member can be granted access to specific work types (Budget, Contracts, Supply Orders, etc.). Access is scoped per work type - a user with Budget access does not automatically have Contracts access.

  • View: User can view requests for this work type
  • Edit: User can create, edit, and submit requests for this work type
  • Dept Head: Informational flag to identify department leadership
Note about Division Membership

Users with division-level access (e.g., Division Heads) automatically have access to all departments in that division. Those memberships are managed from the {% if department.division %} {{ department.division.name }} division page. {% else %} division admin pages. {% endif %}
