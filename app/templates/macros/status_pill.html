{# templates/macros/status_pill.html #} {# Renders a status pill using the engine-shared status → class mapping. Single source of truth for status colors across both item-level and line-level status rendering. Templates that previously rolled their own inline {% if status == 'APPROVED' %}pill-x{% elif ... %} chains should prefer this macro so a one-place edit changes the whole app. Status → pill class: - APPROVED → pill-success (vivid green, line-level) - APPROVED_NEEDS_REVIEW → pill-warning (amber, line-level) - REJECTED → pill-rejected (vivid red, line-level) - NEEDS_INFO → pill-warning (vivid yellow + dark text) - NEEDS_ADJUSTMENT → pill-warning - UNDER_REVIEW → pill-warning (item-level) - PAUSED → pill-warning (item-level) - AWAITING_DISPATCH → pill-warning (item-level) - FINALIZED → pill-approved (subtle green, terminal "settled" state) - SUBMITTED → pill-submitted (subtle blue, in-flight state) - else (DRAFT/PENDING/unknown) → pill-draft (gray) Unknown statuses fall through to pill-draft so new worktypes can introduce statuses without breaking existing renders. Usage: {% from "macros/status_pill.html" import render_status_pill %} {{ render_status_pill(work_item.status) }} {{ render_status_pill(line.status, label=friendly_status(line.status)) }} {{ render_status_pill(status, extra_style="margin-left: 8px;") }} {{ render_status_pill(status, extra_class="line-status-pill") }} # for JS targeting #} {% macro render_status_pill(status, label=None, extra_style=None, extra_class=None) -%} {%- set status_upper = (status or "DRAFT")|upper -%} {%- set pill_class = "pill-success" if status_upper == "APPROVED" else "pill-warning" if status_upper == "APPROVED_NEEDS_REVIEW" else "pill-rejected" if status_upper == "REJECTED" else "pill-warning" if status_upper in ["UNDER_REVIEW", "NEEDS_INFO", "NEEDS_ADJUSTMENT", "PAUSED", "AWAITING_DISPATCH"] else "pill-approved" if status_upper == "FINALIZED" else "pill-submitted" if status_upper == "SUBMITTED" else "pill-draft" -%} {%- set display_label = label if label is not none else status_upper -%} {{ display_label }} {%- endmacro %}
